## q2.py
以下のメンバを持つStudentクラスを作成しろ。
- 学籍番号 id
- 名前 name
- 年齢 age(整数)
- 上記のプロパティをすべて表示する関数showData()

初期化メソッドを用いてインスタンスを生成し、showData()メソッドを呼ぶこと。

```
例)
s = Student("15fi000", "DendaiTaro", 22)
s.showData()

出力結果:
15fi000
DendaiTaro
22
```
